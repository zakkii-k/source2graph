## Phase 6: MCPサーバー

### 追加ファイル

| ファイル | 内容 |
|---|---|
| `src/mcp/graph-store.ts` | グラフのインメモリ管理・ディスクキャッシュ（graph.json）|
| `src/mcp/tools.ts` | 5ツールの実装ロジック |
| `src/mcp/resources.ts` | 3リソースの実装ロジック |
| `src/mcp/server.ts` | McpServer / StdioServerTransport のセットアップ |
| `src/cli/serve.ts` | `codenexus serve` コマンド |

### MCPツール一覧

| ツール | 説明 |
|---|---|
| `analyze` | 指定パスを解析してグラフを構築・キャッシュ |
| `query_nodes` | ラベル / 名前パターン / ファイルパスでノード検索 |
| `get_callers` | 指定シンボルを呼び出しているノードを取得（depth対応）|
| `get_callees` | 指定シンボルが呼び出すノードを取得（depth対応）|
| `get_context` | シンボルの360度ビュー（コンテナ/メソッド/フィールド/継承/呼び出し）|

### MCPリソース一覧

| URI | 内容 |
|---|---|
| `codenexus://graph/schema` | ノードラベル・リレーション型・プロパティ定義 |
| `codenexus://graph/stats` | ラベル/タイプ別カウント・解析日時 |
| `codenexus://graph/nodes/{label}` | 指定ラベルの全ノード |

### 動作確認

```bash
codenexus serve --repo ./test/fixtures/java-sample
# → [codenexus] Done: 33 nodes, 45 rels
# → [codenexus] MCP server ready (stdio)

```

### Claude Code / Cursor での利用方法

`.claude/mcp.json` または `~/.cursor/mcp.json` に以下を追加:

```json
{
  "mcpServers": {
    "codenexus": {
      "command": "npx",
      "args": ["tsx", "/path/to/codenexus/src/cli/index.ts", "serve", "--repo", "/path/to/your/project"]
    }
  }
}
```
